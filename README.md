# Такиев Финанс — Уебсайт

Официален уебсайт на **Такиев Финанс** — счетоводна компания, предлагаща професионални финансови и счетоводни услуги.

## За проекта

Уебсайтът е изграден с фокус върху бързодействие, достъпност и SEO оптимизация. Осигурява на клиентите лесен достъп до информация за предлаганите услуги, публикувани статии и директна връзка с екипа.

## Функционалности

- Представяне на счетоводни и финансови услуги
- Блог с експертни статии
- Видео съдържание
- Контактна форма
- Responsive дизайн за всички устройства
